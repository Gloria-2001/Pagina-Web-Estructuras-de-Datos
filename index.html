Inicio

¿Qué tratamos de hacer?


Sabemos que al iniciar tu camino en una carrera de programación puede resultar muy complicado entender todo a la primera. Por esta razón, se creo esta página web, la cual nos va a ayudar a entender mejor cómo funcionan las estructuras de datos, tales como la Pila, la Cola y/o Listas

Empieza a Aprender


Como ya dijimos, trataremos de ayudarte a comprender mejor el funcionamiento de las principales estructuras de datos: Pila, Cola y Lista.


Empezaremos con las Pilas.


Pila

¿Qué es una Pila?


Se le conoce como una estructura de datos LIFO (last-in, first-out). Es una colección ordenada de elementos en la que se pueden insertar y eliminar por un extremo llamado tope.


Operaciones


Sus operaciones son:

  • Insertar elementos
  • Eliminar elementos
  • Determinar si la pila está llena
  • Determinar si la pila está vacía

Ejemplos cotidianos


En la vida diaria, podemos observar las pilas en los siguientes momentos:

  • Apilar libros
  • Torres de Hanoi
  • Sistema de estacionamiento
  • Leer una secuencia de caracteres

Ahora, continuamos con Colas.


Cola

¿Qué es una Cola?


Se conoce como una estructura FIFO (first-in, first-out). Es una colección ordenada de elementos de la que se pueden borrar elementos en un extremo (llamado el frente de la cola) o insertarlos por el otro (llamado el final de la cola). El primer elemento insertado en la cola, es el primero en ser eliminado


Operaciones


Sus operaciones son:

  • Insertar elemento
  • Eliminar elemento
  • Determinar si la cola está llena
  • Determinar si la cola está vacía

Ejemplos cotidianos


Algunas situaciones de la vida diaria que se asemejan al comportamiento de esta estructura son:

  • Cola de clientes en el banco
  • Parada de camiones
  • Cola en un supermercado
  • Cola en una tortillería

Seguimos con Listas.


Lista

¿Qué es una Lista?


Es una colección ordenada de elementos en la que se pueden insertar y eliminar elementos en el lugar que se requiera. Aunque normalmente se crean listas ordenadas.


Operaciones


Sus operaciones son:

  • Agregar un nodo a la lista
  • Buscar un nodo en la lista
  • Imprimir toda la lista
  • Eliminar un nodo de la lista
  • Modificar los datos de un nodo
  • escribir datos al archivo
  • Leer datos del archivo

Ejemplos cotidianos


A diario, las listas se pueden ver reflejadas en los siguientes ejemplos:

  • Lista para representar el inventario de productos en un supermercado
  • Lista que refleja una cartera de clientes
  • Lista para mostrar ganancias
  • Lista de víveres

Cabe aclarar que existen varios tipos de lista. Aquí solo se mostrará la simple.
